Swift Package Manager Test, creating a command line application, based on the [Ray Wenderlich tutorial](https://www.raywenderlich.com/750-an-introduction-to-the-swift-package-manager).


## Building and Running


**In the Terminal:**


* To compile:
 
```
cd Flag
swift build
```

* To Run:
 
```
cd .build/debug/Flag
```
